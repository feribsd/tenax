### tenax is an larbs insprited tool for void linux featuring some pretty cool stuff such as

- a preconfigured dwl
- iwd
- custom firefox with betterfox user.js and some extensions such as ublock origin vimium etc.
- limine
- custom kernel
- and some more opinonated bullshit made for me to love and enjoy

to run it its simple as running

```bash
curl -sL https://raw.githubusercontent.com/feribsd/tenax/main/tenax.sh | bash
```
naturally dont forget to update the system since void ships a pretty old version of the iso I plan to ship my own iso eventually but for now this will do
